详细描述

Analyzer 是 MgxParser 的核心组件之一，用于对录像文件进行实际的解 析。它会读取帝国时代录像文件中的原始数据并进行必要的处理。Record 类的对象会将 录像文件交给 Analyzer 进行解析，然后将 Analyzer 中得到的数据加工成最后的输出 结果。例如：把表示民族的数字转换成文字。
