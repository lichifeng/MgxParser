virtual

运行解析流程，并返回解析状态。这个函数会修改 valid 的值，让使用者能够通过其了解解释器的状态。比如： 

SomeAnalyzer a = SomeAnalyzer('/path/to/test.mgx');
if(a.valid) {
    dosomething();
}
返回
true 完成了一次有效的解析（即使只读取了部分信息）
false 解析失败（没有获得任何有价值的信息）
